iPhone Shortcut Setup

iOS does not allow a website to silently install a shortcut. Tap the button below to open Shortcuts, then build this once.

Open Shortcuts App

Build the shortcut

  1. Create a new shortcut named Mass Reach Sender.
  2. Add Find Notes:
    • Filter: Name contains MASS GM
    • Sort: Last Modified (Latest First)
    • Limit: 1
  3. Add Get Details of Notes → Detail: Body.
  4. Add Split Text by New Lines, then set variable gmLines.
  5. Add another Find Notes:
    • Filter: Name contains FINAL NUMBERS
    • Sort: Last Modified (Latest First)
    • Limit: 1
  6. Add Get Details of Notes → Body.
  7. Add Split Text by New Lines, then set variable numberLines.
  8. Add Repeat with each item in numberLines:
    • Get random item from gmLines → variable messageText
    • Use Reach app action (if available) to send messageText to Repeat Item
    • If Reach has no shortcut action, use Open App (Reach) and paste/send manually

Only message people who opted in and follow carrier/app anti-spam rules.
